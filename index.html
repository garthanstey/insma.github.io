Our projects

MXA Framework

MXA Framework is an open-source XNA like based framework based on SharpDX Direct3D. It allow you to develop high performance game, 2D and 3D graphics rendering with familiar API.

MXA Game Studio

Visual Studio extension allows to use MXA Framework. It also support standard XNA 4.0 projects.
